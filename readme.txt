== Description ==

Simple social media profile widget with Font Awesome icons.

== Installation ==

1. Upload `naked-social-widget` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Adjust the settings in Settings -> Naked Social Widget
1. Add the Naked Social Widget to your sidebar in Appearance -> Widgets.
